ENGLISH

With these pre-config files your PowKiddy x18s will run GameCube games a lot smoother with less stutters:

1. Install Dolphin MMJR2 latest version
https://github.com/Bankaimaster999/Dolphin-MMJR2/releases ;

2. Copy Dolphin.ini and GFX.ini to the folder mmjr-revamp/config and choose to replace;

3. Open Dolphin MMJR2, re-scan your games, re-map your controls and it’s time to play!

PRO Tip: Use PAL (Europe) game versions because 50hz is lighter to emulate;

PORTUGUÊS

Com esses arquivos de pré-configuração seu PowKiddy x18s vai rodar jogos de GameCube com menos travadas:

1. Instale Dolphin MMJR2
https://github.com/Bankaimaster999/Dolphin-MMJR2/releases ;

2. Copie Dolphin.ini e GFX.ini para a pasta mmjr-revamp/config e escolha substituir;

3. Abra o Dolphin MMJR2, re-escaneie seus roms, re-mapeie seus controles e é hora de jogar!

PRO Tip: Use jogos na versão PAL (Europe) pois 50hz é mais leve de emular;
