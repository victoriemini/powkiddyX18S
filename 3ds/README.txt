ENGLISH

With these pre-config files your PowKiddy x18s will run 3DS games a lot smoother with less stutters:

1. Install Citra MMJ latest version
https://github.com/weihuoya/citra/releases

2. Copy config-mmj.ini to the folder citra-emu/config and choose to replace;

3. Open Citra MMJ and it’s time to play!

PRO Tip: Use PAL (Europe) game versions because 50hz is lighter to emulate;

___

PORTUGUÊS

Com esses arquivos de pré-configuração seu PowKiddy x18s vai rodar jogos de 3DS com menos travadas:

1. Instale Citra MMJ
https://github.com/weihuoya/citra/releases

2. Copie config-mmj.ini para a pasta citra-emu/config e escolha substituir;
